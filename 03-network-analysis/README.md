# Network Analysis

## Objective
To analyze network traffic and identify potential security risks.

## Tools Used
- Nmap
- Wireshark

## Methodology
Performed network scanning and packet capture to identify open ports and suspicious activity.

## Findings
- Port 80 (HTTP) open
- Port 22 (SSH) open

## Threat Analysis
Open ports may allow attackers to exploit vulnerabilities in running services.

## Recommendation
- Restrict access using firewall rules
- Disable unused services
