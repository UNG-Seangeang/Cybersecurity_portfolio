# Incident Response Report

## Incident Summary
A user clicked a phishing email, leading to malware installation.

## Timeline
- 10:00 AM: Email received
- 10:05 AM: Link clicked
- 10:10 AM: Malware executed

## Attack Type
Phishing attack

## Actions Taken
- Isolated affected system
- Removed malware
- Reset user credentials

## Lessons Learned
- Improve user awareness training
- Implement email filtering
